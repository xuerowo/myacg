網頁程式設計 期中教學文件

第一章：網頁設計基本概念

1-1 網站基本概念

  • 全球資訊網 (WWW): 使用超文本 (Hypertext) 技術，透過 HTTP 等協定在網際網路上提供多媒體服務。
  • 網站 (Website) 與網頁 (Web Page): 網站是多個網頁的集合，首頁 (Homepage) 是入口，其他頁面稱為網頁。
  • 網站設計流程: 規劃 -> 設計製作 -> 測試發行 -> 流量分析。
  • 網頁運作: 瀏覽器發送請求 -> 伺服器回傳 HTML -> 瀏覽器解析顯示。
  • 響應式網頁設計 (RWD): 網頁能自動適應不同裝置螢幕大小。
  • 一頁式網站: 所有內容顯示在單一長頁面。

1-2 網頁設計程式語言

  • 瀏覽器端 (Client-Side):
    • 標籤語言: HTML (結構), CSS (樣式)。
    • 程式語言: JavaScript (互動), DHTML (動態效果)。
    • 資料描述: XML。
  • 伺服器端 (Server-Side): PHP, ASP.NET, JSP, CGI (通常用 Perl/C)。在伺服器執行，產生動態內容。

第一章 隨堂測驗

  1. 什麼是響應式網頁設計 (RWD)？它的主要目的是什麼？
  2. 請簡單說明瀏覽器如何取得並顯示一個網頁的過程。
  3. HTML 和 CSS 在網頁設計中各自扮演什麼角色？ (選擇題)
    A) HTML負責互動，CSS負責結構
    B) HTML負責樣式，CSS負責結構
    C) HTML負責結構，CSS負責樣式
    D) HTML負責伺服器邏輯，CSS負責客戶端邏輯
答案
  1. RWD 是一種讓網頁內容能隨不同裝置的解析度自動調整畫面呈現的技術。主要目的是提升使用者在不同裝置上的視覺體驗與操作便利性。
  2. 瀏覽者透過瀏覽器輸入網址 (URL) 或點擊連結，向網頁伺服器發出 HTTP 請求。伺服器找到對應的 HTML 文件後，將其傳回給瀏覽器。瀏覽器接收到 HTML 文件後，解析其中的標籤和內容，並根據 CSS 樣式渲染成使用者看到的網頁畫面。
  3. C) HTML負責結構，CSS負責樣式

第二章：HTML 基本概念

2-1 認識 HTML

  • HTML5: 最新的 HTML 標準，簡化語法，增加新標籤 (如 <header>, <nav>, <audio>, <video>) 和 API，提升網頁功能與互動性，減少對外掛的依賴。
  • W3C 驗證工具: 可用來檢查 HTML 程式碼是否符合標準 (validator.w3.org)。

2-2 HTML 檔案命名與管理

  • 命名原則: 使用半形英數字、小寫、避免空格、加副檔名 (.html 或 .htm)、首頁常命名為 index.html。
  • 資料夾管理: 將所有網站檔案放在一個主資料夾，內部可再依類型 (如 images, css, js) 建立子資料夾。
  • 檔案路徑:
    • 絕對路徑: 完整的 URL (如 https://www.example.com/images/1.png)。
    • 文件相對路徑: 相對於目前 HTML 檔案的位置 (如 images/1.png, ../css/style.css)。
    • 網站根目錄相對路徑: 相對於網站根目錄 (以 / 開頭，如 /images/1.png)。

2-3 HTML 的基本結構

  • 元素 (Element): 由開始標籤、內容、結束標籤組成 (如 <p>這是段落</p>)。
  • 標籤 (Tag): 用 < > 包圍的指令 (如 <p>, </p>)。
  • 屬性 (Attribute): 在開始標籤中提供額外資訊 (如 <img src="1.png" alt="圖片">)。
  • 巢狀元素 (Nesting): 元素可以包含其他元素。
  • 空元素 (Empty Element): 沒有內容，通常沒有結束標籤 (如 <br>, <img>, <hr>)。
  • 全域屬性: 可用於任何 HTML 元素的屬性 (如 id, class, style, title)。

基本網頁結構範例

<!DOCTYPE html> <!-- 宣告為 HTML5 文件 -->
<html lang="zh-Hant-Tw"> <!-- 根元素，設定語言為繁體中文 -->
<head> <!-- 文件標頭，包含元資訊 -->
    <meta charset="UTF-8"> <!-- 設定字元編碼 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- RWD 設定 -->
    <meta name="description" content="網頁描述"> <!-- 網頁描述 -->
    <title>網頁標題</title> <!-- 顯示在瀏覽器標籤上的標題 -->
    <link rel="stylesheet" href="style.css"> <!-- 連結外部 CSS -->
</head>
<body> <!-- 文件主體，顯示在瀏覽器視窗的內容 -->
    <h1>主要標題</h1>
    <p>這是一個段落。</p>
    <script src="script.js"></script> <!-- 連結外部 JavaScript -->
</body>
</html>
(此處無法直接預覽完整網頁，但結構如上所示)

2-4 文件資料元素 (常用於 <head>)

  • <meta>: 提供元數據 (編碼 charset, 描述 description, 關鍵字 keywords, 視口 viewport)。
  • <base>: 設定頁面相對路徑的基準 URL (href) 和預設連結目標 (target)。
  • <link>: 連結外部資源，最常用於 CSS (rel="stylesheet")。
  • <style>: 嵌入內部 CSS 樣式。
  • <script>: 嵌入或連結 JavaScript。src 連結外部檔案，async/defer 控制載入與執行時機。
  • <noscript>: 當 JavaScript 被禁用時顯示的內容。
  • HTML 註解: <!-- 這是註解 -->。

第二章 隨堂測驗

  1. HTML 檔案命名的主要原則有哪些？為什麼首頁通常命名為 `index.html`？
  2. `<head>` 和 `<body>` 標籤在 HTML 文件中的主要功能分別是什麼？
  3. 以下哪個 `<meta>` 標籤用於設定網頁的字元編碼？ (選擇題)
    A) `<meta name="encoding" content="UTF-8">`
    B) `<meta charset="UTF-8">`
    C) `<meta http-equiv="content-type" content="text/html; charset=UTF-8">`
    D) B 和 C 都可以
  4. 相對路徑 `../images/logo.png` 代表什麼意思？
答案
  1. 原則：使用半形英數字、小寫、避免空格、加副檔名。因為伺服器預設會尋找名為 `index.html` (或類似名稱) 的檔案作為網站的入口頁面（首頁）。
  2. `<head>` 包含網頁的元資訊，如標題、編碼、外部資源連結等，這些內容通常不直接顯示在瀏覽器主視窗。`<body>` 包含網頁的主要可見內容，如文字、圖片、連結等。
  3. D) B 和 C 都可以。`<meta charset="UTF-8">` 是 HTML5 的簡化寫法，`<meta http-equiv="content-type" ...>` 是早期寫法，兩者都能設定編碼，但推薦使用 B。
  4. `../` 代表回到上一層目錄，所以這個路徑代表從目前檔案所在的目錄回到上一層，然後進入 `images` 資料夾，找到 `logo.png` 這個檔案。

第三章：常用的 HTML 元素

3-1 段落與列表元素

  • 標題: <h1> 到 <h6>。
  • 段落: <p>。
  • 換行: <br>。
  • 無序列表: <ul> (項目符號)，內含 <li> (列表項目)。
  • 有序列表: <ol> (編號)，內含 <li>。可使用 `type` (編號樣式) 和 `start` (起始值) 屬性。
  • 定義列表: <dl>，內含 <dt> (名詞) 和 <dd> (解釋)。
  • 引述區塊: <blockquote> (通常縮排)。
  • 預格式化: <pre> (保留空格和換行)。
  • 水平線: <hr>。

3-2 文字樣式元素

  • 粗體: <b>, <strong> (語意強調)。
  • 斜體: <i>, <em> (語意強調)。
  • 刪除線: <s>, <del> (表示刪除)。
  • 底線: <u> (少用)。

3-4 超連結元素 (`<a>`)

  • `href` 屬性: 指定目標 URL。

3-5 圖片元素

  • <img>: 嵌入圖片。
    • `src` 屬性: 圖片來源路徑 (必要)。
    • `alt` 屬性: 替代文字 (必要，用於無障礙和圖片失效時)。
    • `width`, `height` 屬性: 設定寬高。
  • <picture>: 提供多個圖片來源，讓瀏覽器依條件選擇。內含 <source> (指定條件和來源) 和一個 <img> (預設)。

圖片範例

<!-- 基本圖片 -->
<img src="1.png" alt="範例圖片一" width="200">

<!-- 使用 picture 提供不同尺寸圖片 -->
<picture>
  <source media="(min-width: 650px)" srcset="2.png"> <!-- 大螢幕用 2.png -->
  <source media="(min-width: 465px)" srcset="1.png"> <!-- 中螢幕用 1.png -->
  <img src="1.png" alt="範例圖片" style="width:auto;"> <!-- 小螢幕或預設用 1.png -->
</picture>

(預覽使用佔位圖片，實際效果需調整瀏覽器寬度查看)

第三章 隨堂測驗

  1. `<ul>` 和 `<ol>` 的主要區別是什麼？它們都需要搭配哪個標籤來定義列表項目？
  2. `<strong>` 和 `<b>` 都可以讓文字變粗，它們在使用上有何建議的區別？
  3. 在 HTML5 中，`<header>`, `<nav>`, `<main>`, `<footer>` 這些標籤被稱為什麼？使用它們有什麼好處？
  4. `<img>` 標籤中的 `alt` 屬性有什麼作用？為什麼它很重要？
答案
  1. `<ul>` (Unordered List) 建立無序列表（項目符號），`<ol>` (Ordered List) 建立有序列表（編號）。它們都需要搭配 `<li>` (List Item) 標籤來定義每個列表項目。
  2. `<strong>` 具有語意上的強調重要性，而 `<b>` 僅表示視覺上的粗體，沒有額外的語意重要性。建議在需要強調內容重要性時使用 `<strong>`，僅需視覺粗體時才考慮 `<b>` (或最好使用 CSS)。
  3. 這些被稱為「語意結構區塊元素」(Semantic Elements)。使用它們的好處是讓 HTML 結構更清晰、更具意義，有助於 SEO（搜尋引擎理解網頁結構）和無障礙性（螢幕閱讀器能更好地解析頁面）。
  4. `alt` (Alternative text) 屬性提供圖片的文字描述。作用：1. 當圖片因故無法載入時，顯示替代文字。 2. 供螢幕閱讀器讀取，幫助視覺障礙者理解圖片內容。 3. 有助於搜尋引擎理解圖片內容。因此它對於網頁可訪問性 (Accessibility) 和 SEO 都很重要。

第四章：影音多媒體、表格及表單元素

4-1 影音多媒體元素

  • <audio>: 嵌入音訊。常用屬性: `src`, `controls`, `autoplay`, `loop`, `preload`。
  • <video>: 嵌入視訊。常用屬性同 `audio`，外加 `width`, `height`, `poster` (預覽圖)。
  • <source>: 在 `audio` 或 `video` 內指定多個來源格式。
  • <iframe>: 嵌入外部網頁或內容 (如 YouTube 影片、Google 地圖)。常用屬性: `src`, `width`, `height`, `frameborder`。
  • <canvas>: 繪圖畫布，需搭配 JavaScript 繪圖。常用屬性: `id`, `width`, `height`。

4-3 表單元素 (`<form>`)

  • <form>: 表單容器。屬性: `action` (提交位址), `method` (`get`/`post`)。
  • <label>: 標籤。`for` 屬性對應 input 的 `id`。
  • <input>: 輸入欄位。`type` 屬性決定類型 (text, password, radio, checkbox, submit, file, date, number, email 等)。常用屬性: `name`, `value`, `placeholder`, `required`, `disabled`。
  • <textarea>: 多行文字輸入。屬性: `rows`, `cols`。
  • <select>: 下拉選單。內含 <option> (選項)。`option` 可有 `value`, `selected` 屬性。
  • <fieldset>: 群組相關欄位。
  • <legend>: `fieldset` 的標題。
  • <output>: 顯示結果。

    </div>
    <div class="mb-4">

      <input type="email" id="email" name="email" required placeholder="請輸入 Email" class="border rounded p-2 w-full">
    </div>

第四章 隨堂測驗

  1. 如何在 HTML 中嵌入 YouTube 影片？主要使用哪個標籤？
  2. 表格的 `colspan="2"` 屬性代表什麼意思？
  3. 在表單中，`radio` 和 `checkbox` 類型的 `<input>` 有何不同？
  4. 表單的 `method` 屬性設定為 `post` 和 `get` 有什麼主要差異？哪種比較安全？
答案
  1. 主要使用 `<iframe>` 標籤。通常可以直接從 YouTube 影片頁面複製嵌入程式碼。
  2. `colspan="2"` 代表該儲存格（`<td>` 或 `<th>`）會水平橫跨 2 個欄位。

第五章：CSS 基本概念

5-1 認識 CSS

  • CSS (Cascading Style Sheets): 層疊樣式表，用於定義 HTML 元素的外觀和排版。
  • 基本語法: `選擇器 { 屬性: 值; 屬性: 值; }`。
  • 註解: `/* 註解 */`。

5-2 CSS 使用方式

  • 行內樣式 (Inline): `<p style="color: red;">...</p>`。
  • 內部樣式表 (Internal): 在 `<head>` 中使用 `<style>` 標籤。
  • 外部樣式表 (External): 寫在獨立的 `.css` 檔，用 `<link>` 引入。

樣式優先權 (Specificity & Cascade)

  • 來源優先級: `!important` > 行內 > ID > Class/屬性/偽類 > 標籤/偽元素 > 通用 > 瀏覽器預設。
  • 相同優先級: 後定義的樣式會覆蓋先定義的。

5-3 CSS 基本選擇器

  • 標籤選擇器: `p { ... }`
  • Class 選擇器: `.classname { ... }`
  • ID 選擇器: `#idname { ... }` (ID 應唯一)
  • 通用選擇器: `* { ... }`
  • 屬性選擇器: `[attribute]`, `[attribute="value"]`, `[attribute^="value"]`, `[attribute$="value"]`, `[attribute*="value"]`。

基本選擇器範例

/* 標籤選擇器 */
p {
  color: gray;
}

/* Class 選擇器 */
.highlight {
  background-color: yellow;
  font-weight: bold;
}

/* ID 選擇器 */
#main-title {
  font-size: 2em;
  color: navy;
}

/* 屬性選擇器 (選擇所有 type="text" 的 input) */
input[type="text"] {
  border: 1px solid blue;
}

/* 屬性選擇器 (選擇 href 以 https 開頭的 a 標籤) */
a[href^="https://"] {
  color: green;
}

第五章 隨堂測驗

  1. 請說明 CSS 的三種使用方式及其優先級。
  2. ID 選擇器和 Class 選擇器的主要區別是什麼？
  3. CSS 選擇器 `div > p` 和 `div p` 有何不同？
  4. `:hover` 和 `::before` 分別屬於哪種類型的選擇器？它們的作用是什麼？
答案
  1. 三種方式：行內樣式 (Inline)、內部樣式表 (Internal)、外部樣式表 (External)。優先級：行內 > 內部/外部 (後載入優先) > 瀏覽器預設。若考慮 `!important`，則其優先級最高。
  2. ID 選擇器 (`#idname`) 用於選取頁面上唯一的元素，具有較高的優先級。Class 選擇器 (`.classname`) 可以套用至頁面上多個元素，優先級低於 ID。
  3. `div > p` (子選擇器) 只會選取作為 `<div>` 直接子元素的 `<p>` 標籤。`div p` (後代選擇器) 會選取 `<div>` 元素內部的所有 `<p>` 標籤，無論層級深度。
  4. `:hover` 屬於虛擬類別選擇器 (Pseudo-class)，用於定義元素在滑鼠指標懸停在其上時的樣式。`::before` 屬於虛擬元素選擇器 (Pseudo-element)，用於在選定元素的內容之前插入一些生成內容 (需搭配 `content` 屬性)。

第六章：CSS 樣式

6-1 色彩屬性

  • 色彩值: 名稱, Hex (`#RRGGBB`), RGB/RGBA, HSL/HSLA。
  • `color` 屬性: 設定文字顏色。

6-2 字型屬性 (`font-`)

  • `font-family`:** 設定字體 (可設多個備用)。
  • `font-size`:** 設定大小 (px, %, em, rem, 關鍵字)。
  • `font-style`:** normal, italic, oblique。
  • `font-variant`:** normal, small-caps。
  • `font-weight`:** normal, bold, bolder, lighter, 100-900。

6-3 文字段落屬性 (`text-`, `line-`, `letter-`)

  • `text-align`:** 水平對齊 (left, right, center, justify)。
  • `text-decoration`:** 文字裝飾線 (underline, overline, line-through)。
  • `text-shadow`:** 文字陰影。
  • `line-height`:** 行高。
  • `letter-spacing`:** 字元間距。

字型與文字樣式範例

.text-sample {
  color: darkslateblue; /* 文字顏色 */
  font-family: "Microsoft JhengHei", "PingFang TC", sans-serif; /* 字體 */
  font-size: 18px; /* 文字大小 */
  font-weight: bold; /* 粗體 */
  line-height: 1.6; /* 行高 */
  text-align: justify; /* 左右對齊 */
  text-shadow: 1px 1px 2px lightgray; /* 文字陰影 */
}

.text-sample a {
  text-decoration: none; /* 移除連結底線 */
  color: teal;
}

.text-sample a:hover {
  text-decoration: underline; /* 滑鼠懸停時加底線 */
}

6-4 清單屬性 (`list-style-`)

  • `list-style-type`:** 符號類型 (disc, circle, square, decimal, none 等)。
  • `list-style-position`:** 符號位置 (inside, outside)。
  • `list-style-image`:** 使用圖片符號 (`url(...)`)。
  • `list-style`:** 縮寫。

6-5 邊框屬性 (`border-`)

  • `border`:** 縮寫 (寬度 樣式 顏色)。
  • `border-width`:** 寬度。
  • `border-style`:** 樣式 (solid, dotted, dashed, double 等)。
  • `border-color`:** 顏色。
  • `border-radius`:** 圓角。
  • `border-image`:** 圖片邊框。

6-6 背景屬性 (`background-`)

  • `background-color`:** 背景顏色。
  • `background-image`:** 背景圖片 (`url(...)`)。
  • `background-repeat`:** 重複方式 (repeat, no-repeat, repeat-x, repeat-y)。
  • `background-position`:** 圖片位置 (center, top, bottom, left, right, 數值)。

6-7 漸層函式

CSS 漸層可以在背景中創造平滑的顏色過渡效果，不需要使用圖片。主要有線性漸層和徑向漸層。

  • 線性漸層 (`linear-gradient()`): 顏色沿直線方向變化。
    • 語法: `background-image: linear-gradient(方向, 顏色1, 顏色2, ...);`
    • 方向: 可以是角度 (如 `90deg`, `180deg`) 或關鍵字 (如 `to bottom`, `to top right`)。預設是 `to bottom`。
    • 顏色: 可以指定多個顏色，並可加上位置 (百分比或長度)。
  • 徑向漸層 (`radial-gradient()`): 顏色從中心點向外呈圓形或橢圓形擴散。
    • 語法: `background-image: radial-gradient(形狀 大小 at 位置, 顏色1, 顏色2, ...);`
    • 形狀: `circle` (圓形) 或 `ellipse` (橢圓形 - 預設)。
    • 大小: 關鍵字 (`closest-side`, `closest-corner`, `farthest-side`, `farthest-corner`) 或具體長度/百分比。
    • 位置: `at` 後面接位置關鍵字 (`center`, `top`, `left` 等) 或百分比/長度。預設是 `center`。
  • 重複漸層 (`repeating-linear-gradient()`, `repeating-radial-gradient()`): 建立重複的漸層圖案。

漸層背景範例

/* 線性漸層：從左到右，藍到綠 */
.gradient-linear {
  background-image: linear-gradient(to right, steelblue, lightgreen);
}

/* 線性漸層：對角線，紅到黃到綠 */
.gradient-diagonal {
  background-image: linear-gradient(to bottom right, red, yellow, limegreen);
}

/* 徑向漸層：圓形，中心白到外圈藍 */
.gradient-radial {
  background-image: radial-gradient(circle, white, dodgerblue);
}

/* 徑向漸層：橢圓，從左上角開始，黃到橘 */
.gradient-radial-positioned {
   background-image: radial-gradient(ellipse at top left, gold, darkorange);
}

6-8 表格屬性

除了使用 `border` 屬性外，CSS 還提供了一些專門用於美化表格的屬性。

  • `border-collapse`:** 控制表格邊框是合併 (`collapse`) 還是分開 (`separate` - 預設)。合併模式下，相鄰儲存格邊框會合併為單一邊框。
  • `border-spacing`:** 當 `border-collapse` 為 `separate` 時，設定相鄰儲存格邊框之間的距離。可設一個值 (水平和垂直相同) 或兩個值 (水平 垂直)。
  • `caption-side`:** 設定表格標題 (`<caption>`) 相對於表格的位置 (`top` - 預設, `bottom`)。
  • `empty-cells`:** 控制是否顯示內容為空的儲存格的邊框和背景 (`show` - 預設, `hide`)。僅在 `border-collapse: separate;` 時有效。
  • `table-layout`:** 控制表格的佈局算法。
    • `auto` (預設): 欄寬由內容決定。
    • `fixed`: 欄寬由表格寬度和 `width` 屬性決定，通常渲染更快。

表格樣式範例

.styled-table {
  width: 100%;
  border-collapse: collapse; /* 合併邊框 */
  margin-bottom: 1rem;
  table-layout: fixed; /* 固定欄寬 */
}

.styled-table caption {
  caption-side: bottom; /* 標題放底部 */
  margin-top: 0.5rem;
  font-style: italic;
  color: gray;
}

.styled-table th, .styled-table td {
  border: 1px solid lightgray;
  padding: 10px;
  text-align: center; /* 文字置中 */
}

.styled-table th {
  background-color: lightsteelblue;
  color: navy;
}

/* 斑馬條紋效果 */
.styled-table tbody tr:nth-child(even) {
  background-color: #f2f2f2;
}

.styled-table tbody tr:hover {
  background-color: #ddd; /* 滑鼠懸停效果 */
}

  1. 請寫出至少三種在 CSS 中表示顏色的方法。
  2. `font-size` 屬性可以使用哪些單位？`em` 和 `rem` 有何不同？
  3. 如何移除超連結預設的底線？當滑鼠移上去時才顯示底線？

  6. `border-collapse: collapse;` 屬性用在表格上時有什麼效果？
答案
  1. 顏色名稱 (如 `red`)、十六進位碼 (如 `#ff0000`)、RGB/RGBA (如 `rgb(255, 0, 0)`)、HSL/HSLA (如 `hsl(0, 100%, 50%)`)。
  2. 可用單位：`px`, `%`, `em`, `rem`, 關鍵字 (如 `medium`) 等。`em` 是相對於父元素的字體大小，會層疊計算。`rem` (root em) 是相對於根元素 (`<html>`) 的字體大小，計算較單純。
  3. 移除底線：`a { text-decoration: none; }`。滑鼠移上顯示底線：`a:hover { text-decoration: underline; }`。
  4. `cover` 會縮放背景圖片，使其完全覆蓋背景區域，可能會裁切掉圖片的一部分。`contain` 會縮放背景圖片，使其寬度或高度其中一邊剛好符合容器大小，確保整張圖片都可見，但可能無法填滿整個容器。
  5. `background-image: linear-gradient(to bottom, red, blue);` 或 `background-image: linear-gradient(red, blue);` (因為 `to bottom` 是預設值)。
  6. 它會使表格的邊框合併成單一線條，而不是每個儲存格都有獨立的邊框（預設的 `separate` 模式）。
