第二章：HTML 基本概念

第二章 模擬測驗

1. (選擇題) 在 HTML 中，哪個標籤用於定義網頁的主要可見內容？

2. (選擇題) 為了避免網頁出現亂碼，通常在 <head> 中使用哪個 <meta> 標籤來宣告字元編碼？

3. (問答題) 請解釋 HTML 中「相對路徑」和「絕對路徑」的區別，並各舉一個例子。

第二章 參考答案

1. (C) <body>

2. (C) <meta charset="UTF-8">

3.

  • 絕對路徑: 指向資源的完整 URL 位址。例子: https://www.example.com/images/logo.png
  • 相對路徑: 相對於目前文件所在位置的路徑。例子: images/logo.png (表示目前目錄下的 images 資料夾中的 logo.png) 或 ../page.html (表示上一層目錄的 page.html)。

第三章 模擬測驗

2. (選擇題) 哪個 HTML5 語意標籤最適合用來放置網站的主要導覽連結？

第三章 參考答案

1. (D) <ul>

2. (C) <nav>

3. (C) target

4. 主要區別在於語意：<strong> 表示內容具有重要性、嚴重性或緊急性（語意上的強調）；而 <b> 僅表示視覺上的粗體，沒有額外的語意重要性，適用於不需要強調語意的場合（如關鍵字、產品名稱）。

第四章：影音多媒體、表格及表單元素

第四章 模擬測驗

3. (選擇題) 在表單中，要建立一個讓使用者可以輸入多行文字的區域，應該使用哪個標籤？

4. (問答題) 請說明表單中 method 屬性的 get 和 post 兩種傳送方式的主要差異。

第四章 參考答案

1. (C) <iframe>

2. (D) <th>

3. (D) <textarea>

4.

  • GET: 表單資料會附加在 URL 後面傳送，長度有限制，安全性較低，適合用於搜尋或非敏感資料。
  • POST: 表單資料包含在 HTTP 請求的主體中傳送，沒有長度限制，安全性較高，適合用於提交敏感資料或大量資料（如包含檔案上傳）。
