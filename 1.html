網頁程式設計教學

第一章：網頁設計基本概念

1-1 網站基本概念

本節介紹網站、網頁、全球資訊網(WWW)的基本定義，以及網站的規劃、設計流程和運作方式。

  • 全球資訊網 (WWW)：基於超文本(Hypertext)技術，透過HTTP協定在網際網路上提供多媒體資訊服務的系統。
  • 網站 (Website)：由多個相關網頁組成的集合，通常有一個主要的入口頁面，稱為首頁 (Homepage)。
  • 網頁 (Web Page)：構成網站的基本單位，用來展示文字、圖片、影音等資訊。
  • 網站設計流程：
    1. 內容規劃：確定目標、架構。
    2. 設計與製作：視覺設計、HTML/CSS編寫。
    3. 測試與發行：功能測試、上傳伺服器。
    4. 流量分析：監控與優化。
  • 網頁運作原則：使用者透過瀏覽器請求 -> 伺服器回應HTML文件 -> 瀏覽器解析並顯示網頁。
  • 響應式網頁設計 (RWD)：讓網頁能在不同尺寸的裝置（電腦、平板、手機）上都有良好的瀏覽體驗。
  • 一頁式網站：將所有內容放在單一頁面，透過滾動瀏覽，適合內容較少或特定行銷目的的網站。

1-2 網頁設計程式語言

網頁的構成涉及多種語言和技術，主要分為瀏覽器端和伺服器端。

瀏覽器端 (Client-Side)

  • HTML (HyperText Markup Language)：定義網頁的結構和內容。
  • CSS (Cascading Style Sheets)：描述網頁的外觀和樣式。
  • JavaScript：實現網頁的互動效果和動態功能。(DHTML是早期結合HTML, CSS, JavaScript的技術概念)
  • XML (Extensible Markup Language)：用於描述和傳輸資料，與HTML不同，它著重於資料本身而非呈現。

伺服器端 (Server-Side)

這些語言在伺服器上執行，處理業務邏輯、資料庫互動等，然後生成HTML傳送給瀏覽器。

  • PHP
  • ASP.NET
  • JSP (JavaServer Pages)
  • Python (搭配 Django/Flask 框架)
  • Ruby (搭配 Rails 框架)
  • Node.js (使用 JavaScript)
  • CGI (Common Gateway Interface)：一種較早期的伺服器與外部程式溝通的標準。

MVC 架構

一種常見的軟體設計模式，將應用程式分為三層：

  • Model (模型)：處理資料和業務邏輯。
  • View (視圖)：負責使用者介面的呈現 (通常是 HTML/CSS/JS)。
  • Controller (控制器)：接收使用者請求，調度模型和視圖。

Web API (應用程式介面)

讓不同的應用程式可以互相溝通、交換資料或服務的介面。例如，在地圖上顯示店家位置（使用 Google Maps API）、使用社群帳號登入等。

第二章：HTML 基本概念

2-1 認識 HTML

HTML 是構成網頁的基礎。HTML5 是目前的最新標準，它簡化了語法，增加了許多新功能（如影音支援、繪圖等），並提升了語意化。

可以使用 W3C 的驗證工具 (http://validator.w3.org) 來檢查 HTML 程式碼是否符合標準。

2-2 HTML 檔案命名的原則

  • 副檔名：使用 .html 或 .htm。
  • 字元：建議使用半形小寫英文字母、數字、底線 _ 或連字號 -。避免使用中文、全形字元或空格。
  • 首頁檔名：通常命名為 index.html (或伺服器指定的名稱如 default.htm)。
  • 資料夾管理：將網站所有檔案放在一個根資料夾，並建立子資料夾 (如 images, css, js) 來組織檔案。

檔案路徑

  • 絕對路徑：完整的 URL，如 https://www.example.com/images/logo.png。
  • 相對路徑：相對於目前 HTML 檔案的路徑。
    • images/logo.png：目前資料夾下的 images 資料夾內的 logo.png。
    • ../styles/main.css：上一層資料夾下的 styles 資料夾內的 main.css。
  • 根目錄相對路徑：從網站根目錄開始的路徑，以 / 開頭，如 /images/logo.png。

2-3 HTML 的基本結構

一個基本的 HTML 文件結構如下：

<!DOCTYPE html>
<html lang="zh-Hant-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>網頁標題</title>
    <!-- 其他 head 內容，如 CSS, JavaScript 連結 -->
</head>
<body>
    <!-- 網頁可見內容 -->
    <h1>這是主標題</h1>
    <p>這是一個段落。</p>
</body>
</html>

元素、標籤與屬性

  • 元素 (Element)：由開始標籤、內容和結束標籤組成，如 <p>這是內容</p>。
  • 標籤 (Tag)：用 < 和 > 包圍的關鍵字，如 <p> (開始標籤) 和 </p> (結束標籤)。
  • 空元素 (Empty Element)：沒有內容，只有開始標籤，如 <br>, <img>, <hr>, <input>。可以寫成自我關閉形式，如 <br />。
  • 屬性 (Attribute)：在開始標籤中提供額外資訊，格式為 名稱="值"，如 <img src="images/1.png" alt="圖片描述">。
  • 巢狀元素 (Nested Element)：元素可以包含其他元素，如 <div><p>文字</p></div>。
  • 全域屬性 (Global Attributes)：可以用在幾乎所有 HTML 元素上的屬性，如 id, class, style, title, lang。

主要結構標籤說明

  • <!DOCTYPE html>：宣告文件類型為 HTML5。
  • <html>：根元素，lang 屬性指定頁面語言。
  • <head>：包含頁面的元數據 (metadata)，如標題、編碼、CSS連結等，內容不會直接顯示。
  • <meta charset="UTF-8">：指定字元編碼，防止亂碼。
  • <meta name="viewport" ...>：設定響應式設計的可視區域。
  • <title>：定義瀏覽器標籤頁和書籤中顯示的標題。
  • <body>：包含所有使用者可見的網頁內容。

2-4 文件資料元素 (常用於 <head>)

  • <meta>：提供元數據。
    • name="description"：網頁描述。
    • name="keywords"：網頁關鍵字 (SEO 權重已降低)。
    • name="author"：作者。
    • http-equiv="refresh" content="秒數;url=網址"：自動刷新或跳轉。
  • <base href="基準URL" target="_blank">：設定頁面相對路徑的基準 URL 和預設連結目標。
  • <link rel="stylesheet" href="css/style.css">：連結外部 CSS 樣式表。
  • <style> ...CSS規則... </style>：定義內部 CSS 樣式。
  • <script src="js/script.js"></script>：連結外部 JavaScript 檔案。
  • <script> ...JS代碼... </script>：嵌入內部 JavaScript 代碼。async 和 defer 屬性可控制載入和執行時機。
  • <noscript> ...訊息... </noscript>：在瀏覽器不支援或禁用 JS 時顯示的內容。
  • HTML 註解：<!-- 這是註解 -->，瀏覽器會忽略。

3-3 語意結構區塊元素

HTML5 引入了許多語意化標籤，讓網頁結構更清晰。

  • <div>：通用的區塊容器，無特定語意，常用於佈局和樣式分組。
  • <header>：頁首區域，包含標題、logo、導覽等。
  • <nav>：導覽連結區域。
  • <main>：頁面的主要內容區域（一個頁面通常只有一個）。
  • <section>：文件中的一個獨立區段或主題分組。
  • <article>：獨立、完整的內容單元，如部落格文章。
  • <aside>：側邊欄，放置與主要內容關聯性較低的資訊。
  • <footer>：頁尾區域，包含版權、聯絡方式等。
  • <address>：聯絡資訊。
<body>
    <header>
        <h1>網站標題</h1>
        <nav>
            <a href="#">首頁</a> | <a href="#">關於</a>
        </nav>
    </header>

        <article>
            <h2>文章標題</h2>
            <p>文章內容...</p>
            <section>

3-4 超連結元素: <a>

<a> (Anchor) 元素用於創建超連結。

  • href 屬性：指定連結的目標 URL 或資源。
  • target 屬性：指定如何開啟連結 (_blank 在新視窗/分頁開啟)。
  • download 屬性：提示瀏覽器下載連結的檔案。
  • 錨點連結：href="#elementID" 連結到同一頁面中具有相應 id 的元素。
<!-- 連結到外部網站 -->
<a href="https://www.google.com" target="_blank">前往 Google</a><br>

<!-- 連結到 Email -->
<a href="mailto:info@example.com">寄信給我們</a><br>

<!-- 連結到電話 -->
<a href="tel:+1234567890">撥打電話</a><br>

<!-- 連結到頁內錨點 -->
<a href="#section-target">跳到目標區塊</a><br>

<!-- 圖片連結 -->
<a href="https://www.example.com"><img src="images/1.png" alt="範例圖片" width="50"></a><br>

<!-- 下載連結 -->
<a href="files/document.pdf" download="我的文件.pdf">下載文件</a>

<!-- 頁內錨點目標 -->
<h3 id="section-target" style="margin-top:500px;">這裡是目標區塊</h3>

第四章：影音多媒體、表格及表單元素

4-1 影音多媒體元素

音訊 (Audio): <audio>

嵌入音訊檔案。支援 .mp3, .wav, .ogg 格式。

  • src 或 <source>：指定音訊來源。
  • controls：顯示播放控制器。
  • autoplay：自動播放 (可能被瀏覽器阻止)。
  • loop：循環播放。
  • muted：預設靜音。
  • preload：預載方式 (`none`, `metadata`, `auto`)。
<audio controls>
    <source src="audio/sample.mp3" type="audio/mpeg">
    <source src="audio/sample.ogg" type="audio/ogg">
    您的瀏覽器不支援 audio 元素。
</audio>

第五章：CSS 基本概念

5-1 認識 CSS

CSS (Cascading Style Sheets) 用於定義 HTML 文件的外觀和樣式，實現內容與表現的分離。

CSS 基本架構

/* CSS 註解 */
選擇器 {
    屬性名: 屬性值; /* 宣告 1 */
    屬性名: 屬性值; /* 宣告 2 */
}

/* 範例 */
p {
    color: blue;
    font-size: 16px;
}
  • 選擇器 (Selector)：指定要套用樣式的 HTML 元素。
  • 宣告區塊 (Declaration Block)：用 { } 包裹，包含一或多個宣告。
  • 宣告 (Declaration)：包含一個屬性名和屬性值，用 : 分隔。
  • 分號 (;)：用於分隔多個宣告。
  • 註解：用 /* ... */ 包裹。

5-2 CSS 的使用方式

  1. 行內樣式 (Inline Style)：直接在 HTML 元素的 style 屬性中編寫。優先級最高，但不易維護。 
    <p style="color: red; font-weight: bold;">這是行內樣式。</p>

    結果預覽:

    這是行內樣式。

  2. 內部樣式表 (Internal Style Sheet)：在 HTML 文件的 <head> 中使用 <style> 標籤定義。只對當前頁面有效。 
    <head>
    <style>
        .internal-style {
            color: green;
            text-decoration: underline;
        }
    </style>
</head>
<body>
    <p class="internal-style">這是內部樣式表定義的樣式。</p>
</body>

    結果預覽:

    這是內部樣式表定義的樣式。

  3. 外部樣式表 (External Style Sheet)：將 CSS 規則寫在獨立的 .css 檔案中，透過 <link> 標籤引入。最常用，便於管理和重用。 
    <!-- 在 HTML 的 <head> 中 -->
<link rel="stylesheet" href="styles/main.css">

    background-color: lightyellow;
    border: 1px solid gold;
    padding: 5px;

樣式表的串接順序與優先級

當多個樣式規則應用到同一個元素時，瀏覽器根據以下原則決定最終樣式：

  1. 來源：行內樣式 > 內部樣式 > 外部樣式 > 瀏覽器預設樣式。
  2. !important：宣告中加入 !important 的規則具有最高優先級。
  3. 明確度 (Specificity)：選擇器的特定程度越高，優先級越高 (ID > Class/屬性/偽類 > 標籤/偽元素)。
  4. 順序：在優先級相同的情況下，後定義的規則會覆蓋先定義的。
<!-- HTML -->
<p id="para1" class="text blue" style="color: orange;">哪個顏色優先？</p>

<!-- CSS -->
<style>
    p { color: black; } /* 標籤 */
    .text { color: green; } /* Class */
    .blue { color: blue !important; } /* Class + !important */
    #para1 { color: purple; } /* ID */
</style>

5-3 CSS 基本選擇器

  • 標籤選擇器 (Type Selector)：選擇所有指定 HTML 標籤。例：p { ... }
  • Class 選擇器 (Class Selector)：選擇所有具有指定 class 屬性的元素。例：.highlight { ... }
  • ID 選擇器 (ID Selector)：選擇具有指定 id 屬性的唯一元素。例：#main-nav { ... }
  • 通用選擇器 (Universal Selector)：選擇所有元素。例：* { margin: 0; }
  • 屬性選擇器 (Attribute Selector)：根據屬性或屬性值選擇。
    • [href]：選擇所有帶 href 屬性的元素。
    • [target="_blank"]：選擇 target 屬性值為 "_blank" 的元素。
    • [class~="warning"]：選擇 class 屬性值包含 "warning" 這個詞的元素。
    • [lang|="en"]：選擇 lang 屬性值為 "en" 或以 "en-" 開頭的元素。
    • a[href^="https://"]：選擇 href 以 "https://" 開頭的 a 元素。
    • img[src$=".png"]：選擇 src 以 ".png" 結尾的 img 元素。
    • a[href*="example"]：選擇 href 包含 "example" 的 a 元素。
/* 標籤 */
h2 { font-style: italic; }
/* Class */
.important-note { background-color: yellow; }
/* ID */
#footer { border-top: 1px solid gray; }
/* 屬性 */
input[type="text"] { border: 1px solid blue; }
/* 屬性值包含 */
a[href*="tutorial"] { font-weight: bold; }

5-4 組合選擇器

  • 後代選擇器 (空格)：選擇父元素內的所有後代。例：article p { ... } (選擇 article 內所有 p)
  • 子選擇器 (>)：只選擇父元素的直接子元素。例：ul > li { ... } (只選擇 ul 的第一層 li)
  • 同層相鄰選擇器 (+)：選擇緊接在某元素後的同層元素。例：h3 + p { ... } (選擇緊跟 h3 後的 p)
  • 同層全體選擇器 (~)：選擇某元素之後的所有同層元素。例：h3 ~ p { ... } (選擇 h3 之後所有同層的 p)

6-5 邊框屬性

  • border: 速記屬性 (width style color)。例：border: 2px dashed green;
  • border-width: 寬度 (thin, medium, thick, 單位值)。
  • border-style: 樣式 (solid, dotted, dashed, double, groove, ridge, inset, outset, none)。
  • border-color: 顏色。
  • 可單獨設定各邊：border-top, border-right, border-bottom, border-left。
  • border-radius: 圓角半徑 (px, %)。
  • border-image: 使用圖片作為邊框 (較複雜，包含 source, slice, width, outset, repeat)。
.border-box {
    border: 3px solid tomato;
    padding: 10px;
    margin-bottom: 10px;
}
.rounded-box {
    border: 2px dotted blue;
    border-radius: 15px; /* 圓角 */
    padding: 10px;
    margin-bottom: 10px;
}
.fancy-border {
    border-top: 4px double red;
    border-bottom: 4px double red;
    border-left: 2px solid gray;
    border-right: 2px solid gray;
    padding: 10px;
    margin-bottom: 10px;
}
.image-border-example { /* 需要邊框圖片 images/1.png */
    border: 30px solid transparent;
    padding: 15px;
    border-image-source: url('images/1.png');
    border-image-slice: 30;
    border-image-repeat: round;
}

6-6 背景屬性

  • background-color: 背景顏色。
  • background-image: 背景圖片 (url('path'))。
  • background-repeat: 重複方式 (repeat, no-repeat, repeat-x, repeat-y)。
  • background-position: 背景圖片位置 (關鍵字或單位值)。
  • background-attachment: 背景固定方式 (scroll, fixed, local)。
  • background-size: 背景圖片大小 (auto, cover, contain, 單位值)。
  • background: 速記屬性 (color image repeat attachment position / size)。
.bg-demo {
    width: 300px;
    height: 150px;
    border: 1px solid #ccc;
    margin-bottom: 10px;
    color: white; /* 讓文字可見 */
    text-shadow: 1px 1px 1px black; /* 文字陰影 */
}
.bg-color {
    background-color: lightcoral;
}
.bg-image {
    background-image: url('images/2.png');
    background-repeat: no-repeat;
    background-position: center center;
    background-size: contain; /* 完整顯示圖片 */
}
.bg-fixed {
    height: 200px; /* 需要足夠高度才能看到滾動效果 */
    overflow: auto; /* 添加滾動條 */
    background: lightblue url('images/1.png') no-repeat fixed center;
    background-size: 50px;
    padding: 20px;
    color: black;
    text-shadow: none;
}

6-7 漸層函式 (用於 background-image)

  • 線性漸層 (Linear Gradient): linear-gradient(direction, color1, color2, ...)
  • 放射漸層 (Radial Gradient): radial-gradient(shape size at position, color1, color2, ...)
  • 圓錐漸層 (Conic Gradient): conic-gradient(from angle at position, color1, color2, ...)
  • 重複漸層: repeating-linear-gradient(), repeating-radial-gradient(), repeating-conic-gradient()
.gradient-linear {
    background-image: linear-gradient(to right, steelblue, lightgreen);
}
.gradient-radial {
    background-image: radial-gradient(circle, yellow, orange, red);
}
.gradient-conic { /* 彩虹效果 */
    background-image: conic-gradient(red, yellow, lime, aqua, blue, magenta, red);
    border-radius: 50%; /* 變圓形 */
}
.gradient-repeating { /* 斜條紋 */
    background-image: repeating-linear-gradient(
        45deg,
        #ccc,
        #ccc 10px,
        #eee 10px,
        #eee 20px
    );
}

6-8 表格屬性 (CSS)

  • caption-side: 表格標題位置 (top, bottom)。
  • border-collapse: 邊框合併模式 (separate, collapse)。
  • border-spacing: 分離邊框模式下的間距 (僅用於 border-collapse: separate)。
  • table-layout: 表格佈局算法 (auto, fixed)。
  • empty-cells: 分離邊框模式下是否顯示空儲存格的邊框 (show, hide)。
  • vertical-align: (用於 td, th) 垂直對齊 (top, middle, bottom, baseline)。
<style>
.styled-table {
    width: 400px;
    border-collapse: collapse; /* 合併邊框 */
    table-layout: fixed; /* 固定佈局 */
    margin-bottom: 15px;
}
.styled-table caption {
    caption-side: bottom; /* 標題在下方 */
    margin-top: 5px;
    font-style: italic;
}

    border: 1px solid darkblue;
    padding: 8px;
    text-align: center;

.styled-table th {
    background-color: lightblue;
}
.styled-table td.align-top {
    vertical-align: top; /* 垂直靠上 */
    height: 50px; /* 給予高度看效果 */
}
.styled-table td.empty-cell-demo {
    /* 在 collapse 模式下 empty-cells 無效 */
}
</style>
