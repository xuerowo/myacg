網頁程式設計教學 (CH1-CH6)

第一章：網頁設計基本概念

1-1 網站基本概念

全球資訊網 (WWW)：運用超文本(Hypertext)技術，整合HTTP等通訊協定，在網際網路上提供多媒體服務的系統。

網站與網頁：網站是多個網頁的集合。首頁(Homepage)是入口，網頁(Web Page)是其他頁面。

網站設計流程：

  1. 內容規劃
  2. 設計與製作
  3. 測試與發行
  4. 流量分析

網頁運作原則：瀏覽器(Client)透過HTTP向伺服器(Server)請求網頁，伺服器回傳HTML文件，瀏覽器解析並顯示。

響應式網頁設計 (RWD)：讓網頁能自動適應不同裝置的螢幕尺寸，提供最佳瀏覽體驗。

一頁式網站：所有內容集中在單一頁面，透過滾動瀏覽。

1-2 網頁設計程式語言

瀏覽器端 (Client-Side)

  • HTML：網頁結構與內容。
  • CSS：網頁樣式與外觀。
  • JavaScript (JS)：網頁互動與動態效果 (DHTML是早期概念，現多指HTML+CSS+JS)。
  • XML：描述資料結構。

伺服器端 (Server-Side)

  • ASP.NET：Microsoft 開發框架。
  • PHP：開源腳本語言，廣泛使用。
  • JSP：Java 技術。
  • CGI：通用閘道介面。
  • (還有 Python/Django/Flask, Ruby/Rails, Node.js 等)

MVC 架構

將應用程式分為模型(Model)、檢視(View)、控制器(Controller)，以利於開發與維護。

Web API

應用程式介面，讓不同程式可以透過網路交換資料或服務，例如嵌入 Google Map。

第二章：HTML 基本概念

2-1 認識 HTML

HTML5 是最新的 HTML 標準，簡化語法並加入新功能。

可使用 W3C Validator 驗證 HTML。

2-2 HTML 檔案命名與管理

  • 副檔名：.html 或 .htm。
  • 命名：半形英文/數字，建議小寫，避免空格。
  • 首頁：通常為 index.html。
  • 資料夾管理：將 HTML、CSS、JS、圖片等分門別類。

檔案路徑

  • 絕對路徑：完整 URL，如 https://example.com/images/1.png。
  • 相對路徑：
    • images/1.png (目前目錄下的 images 資料夾)
    • ../styles/main.css (上一層目錄的 styles 資料夾)
  • 根目錄相對路徑：以 / 開頭，相對於網站根目錄，如 /images/1.png。

2-3 HTML 的基本架構

元素 (Element) = 開始標籤 + 內容 + 結束標籤 (如 <p>內容</p>)。

空元素 (Empty Element)：沒有內容和結束標籤 (如 <br>, <img>)。

屬性 (Attribute)：在開始標籤中提供額外資訊 (如 <img src="1.png" alt="範例圖片">)。

巢狀元素 (Nested Element)：元素內可包含其他元素。

網頁基本結構範例

<!DOCTYPE html>
<html lang="zh-Hant-TW">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>網頁標題</title>
        <!-- 連結外部 CSS -->
        <link rel="stylesheet" href="style.css">
        <!-- 內部 CSS -->
        <style>
            /* 這裡可以寫 CSS */
        </style>
    </head>
    <body>
        <!-- 網頁可見內容 -->
        <h1>主要標題</h1>
        <p>這是一個段落。</p>
        <!-- 連結外部 JavaScript -->
        <script src="script.js"></script>
        <!-- 內部 JavaScript -->
        <script>
            // 這裡可以寫 JavaScript
        </script>
    </body>
</html>

2-4 文件資料元素 (Head 內常用)

  • <title>：網頁標題。
  • <meta>：提供元數據。
    • charset="UTF-8"：字元編碼。
    • name="description" content="..."：網頁描述。
    • name="keywords" content="..."：關鍵字。
    • name="author" content="..."：作者。
    • name="viewport" content="..."：行動裝置視口設定。
    • http-equiv="..." content="..."：模擬 HTTP 標頭 (如 `refresh` 跳轉)。
  • <link>：連結外部資源 (最常用於 CSS)。
    • rel="stylesheet"：表示連結的是樣式表。
    • href="路徑/檔名.css"：樣式表路徑。
    • rel="icon" href="favicon.ico"：設定網頁圖示。
  • <style>：嵌入內部 CSS 樣式。
  • <script>：嵌入內部 JavaScript 或連結外部 JS 檔案 (src 屬性)。
    • async：非同步下載並執行 JS (不保證順序)。
    • defer：非同步下載，HTML 解析完畢後按順序執行 JS。
  • <noscript>：當瀏覽器不支援或禁用 JS 時顯示的內容。
  • <base>：設定頁面相對路徑的基準 URL (href) 或預設連結目標 (target)。

第五章：CSS 基本概念

5-1 認識 CSS

CSS (Cascading Style Sheets) 用於設定 HTML 元素的外觀和佈局。

基本架構：選擇器 { 屬性: 值; 屬性: 值; }

/* CSS 註解 */
p {
    color: blue; /* 設定文字顏色為藍色 */
    font-size: 16px; /* 設定字體大小為 16 像素 */
}

5-2 CSS 的使用方式

  • 行內樣式：<p style="color: red;">...</p> (優先權最高，不建議)
  • 內部樣式表：在 <head> 的 <style> 標籤內寫 CSS。
  • 外部樣式表：寫在獨立的 .css 檔案，用 <link> 引入。(推薦)

CSS 串接與優先權

樣式來源優先權：行內 > 內部/外部 > 瀏覽器預設。

!important 宣告具有最高優先權。

選擇器明確度：ID > Class/屬性/虛擬類 > 標籤/虛擬元素 > 通用。

相同優先權下，後定義的規則覆蓋先定義的。

5-3 CSS 基本選擇器

標籤選擇器

h2 { font-family: "Verdana", sans-serif; }

Class 選擇器 (.)

.highlight { background-color: yellow; }
.important { font-weight: bold; }
.highlight.important { color: red; } /* 同時有兩個 class */

這段文字是粗體。

ID 選擇器 (#)

#logo { border: 1px solid blue; }
這是 ID 為 logo 的區塊

通用選擇器 (*)

* { margin: 0; padding: 0; } /* 通常用於重設樣式 */

屬性選擇器 ([])

/* 選擇所有有 title 屬性的 a 標籤 */
a[title] { color: green; }

/* 選擇 type="submit" 的 input 標籤 */
input[type="submit"] { background-color: lightblue; }

/* 選擇 href 以 "https" 開頭的 a 標籤 */
a[href^="https"] { font-style: italic; }

/* 選擇 src 以 ".png" 結尾的 img 標籤 */
img[src$=".png"] { border: 2px solid red; }

/* 選擇 class 包含 "button" 的元素 */
[class*="button"] { cursor: pointer; }

5-4 組合選擇器

後代選擇器 (空格)

article p { line-height: 1.8; } /* article 內所有 p */

子選擇器 (>)

5-6 虛擬元素選擇器 (::)

/* 在 p 元素前插入圖示 */
p.info::before {
    content: "ℹ️ "; /* 必要屬性 */
    margin-right: 5px;
}

/* 設定第一個字母樣式 */
p.first-letter-style::first-letter {
    font-size: 2em;
    color: #0056b3;
    float: left; /* 讓文字環繞 */
    margin-right: 4px;
}

/* 設定第一行樣式 */
p.first-line-style::first-line {
    font-weight: bold;
}

/* 設定列表標記樣式 */
ul.custom-marker li::marker {
    content: "✅ ";
    color: green;
}

/* 設定選取文字樣式 */
::selection {
    background-color: #0056b3;
    color: white;
}

6-2 字型屬性

.font-example {
    font-family: "Georgia", serif; /* 字型 */
    font-size: 18px;           /* 大小 */
    font-style: italic;        /* 樣式: 斜體 */
    font-weight: bold;         /* 粗細: 粗體 */
    font-variant: small-caps;  /* 變化: 小型大寫 */
}
/* 速記寫法: style variant weight size/line-height family */
.font-shorthand {
    font: italic small-caps bold 1.2em/1.5 "Arial", sans-serif;
}

6-5 邊框屬性

.border-example {
    border-width: 2px;              /* 寬度 */
    border-style: dashed;           /* 樣式: 虛線 */
    border-color: green;            /* 顏色 */
    border-radius: 10px;            /* 圓角 */
}
/* 速記寫法 */
.border-shorthand {
    border: 3px solid #0056b3;      /* 寬度 樣式 顏色 */
    border-bottom-color: red;       /* 單獨修改下邊框顏色 */
    border-top-left-radius: 20px;   /* 單獨設定左上圓角 */

/* 圖片邊框 (較複雜，僅示意) */

    border-image-slice: 30;          /* 切割圖片 */
    border-image-repeat: round;      /* 重複方式 */

6-6 背景屬性

.background-example {
    background-color: lightyellow;      /* 背景色 */
    background-image: url('10.png');    /* 背景圖 */
    background-repeat: no-repeat;       /* 不重複 */
    background-position: center center; /* 位置置中 */
    background-size: contain;           /* 調整大小: 包含 */
    background-attachment: scroll;      /* 滾動: 跟隨滾動 */
    padding: 20px; /* 增加內距看效果 */
    height: 150px; /* 設定高度看效果 */
    border: 1px solid #ccc; /* 加邊框區分 */
}
/* 速記寫法 (常用) */
.background-shorthand {
    background: lightblue url('10.png') no-repeat right top / 50px auto;
    /* color image repeat position / size */
    padding: 20px;
    height: 100px;
    border: 1px solid #ccc;
}
