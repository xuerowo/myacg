# 二次元作品推薦網站

這是一個專注於二次元作品推薦的網站，旨在幫助愛好者找到高質量的動畫、網漫、輕小說和視覺小說作品。

## 網站特色

- **動畫推薦**：精選優質動畫作品，包含日本動畫與中國動畫
- **網漫推薦**：熱門網路漫畫推薦，帶你進入精彩的故事世界
- **輕小說推薦**：精選輕小說作品，豐富你的閱讀生活
- **視覺小說推薦**：優質視覺小說遊戲，體驗互動式劇情
- **ACG翻譯**：提供網漫和輕小說的翻譯資源

## 技術實現

- 純HTML/CSS/JavaScript靜態網站
- 響應式設計，適配各種設備
- 支援亮色/暗色主題切換
